8th Grade Extreme Sports Project

The Project

The 8th Grade ELA class at BCMS took on a daunting project at the end of April 2021: Research the topic of extreme sports and create a custom-coded website from scratch to show what you know.

The Process

Each student was given two days to research an extreme sport of their choice and produce an organized slideshow presentation.

The students then learned to code a website using HTML and CSS. The most ambitious and successful projects were collected for this final site.

A Final Note

These pages are best viewed from a computer using Google Chrome or Mozilla Firefox. They will work on mobile devices, but there may be a few visual issues.

Make sure you click the dark gray button to see the code these students wrote by hand!
